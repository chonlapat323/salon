# Salon Hair Brand Master Plan v2

## 1. Project Objective

เป้าหมายของโปรเจกต์นี้คือการสร้างแบรนด์ผลิตภัณฑ์ผมสำหรับร้านซาลอน โดยเริ่มจากกลุ่มสินค้าที่ร้านใช้จริงทุกวันและมีการสั่งซ้ำสูง แล้วต่อยอดไปสู่สินค้ากำไรสูงที่ลูกค้าซื้อกลับบ้านได้

โจทย์หลักของแบรนด์:

- วิเคราะห์ตลาดสีย้อมผมและสินค้าที่เกี่ยวข้อง
- หา model การลงทุนที่เริ่มได้จริง
- ใช้โรงงาน OEM จีนเป็นฐานการผลิต
- วางชื่อแบรนด์และภาพลักษณ์ให้ดู inter แต่ยังเข้าถึงได้
- พัฒนาสินค้าที่มีโอกาสชนะในตลาดร้านซาลอนระดับกลาง

## 2. Market Thesis

### มุมมองตลาดจากล่างขึ้นบน

หากเริ่มวิเคราะห์จากสินค้า `ล่างสุดของ value chain` จะพบว่าตลาดนี้ไม่ได้แข่งกันที่ "สีผม" อย่างเดียว แต่แข่งกันทั้งระบบ ได้แก่:

- สีผม
- ไฮโดรเจนเปอร์ออกไซด์ / developer
- ผงฟอกผม
- เม็ดสีหรือ pigment correction
- แชมพู / ทรีตเมนต์ / คอนดิชันเนอร์
- leave-in และสินค้าดูแลหลังทำสี

แปลว่าแบรนด์ที่ชนะ ไม่จำเป็นต้องเริ่มจากการมีครบทุกอย่าง แต่ต้องเลือกจุดเข้าให้ถูก

### ขนาดโอกาสในเชิงพฤติกรรมตลาด

แม้ในระดับร้านเล็กถึงกลาง สินค้ากลุ่มทำสีมีการใช้ต่อเนื่องทุกเดือน เพราะเป็นหมวดที่เกี่ยวข้องกับ:

- งานปิดผมขาว
- งานแฟชั่นสี
- งานฟอก ยกพื้น และแก้โทน
- งานบำรุงหลังเคมี

ดังนั้นตลาดนี้มี 3 ชั้นของเงิน:

1. `เงินหมุนเร็ว` จากสินค้าใช้ในร้าน เช่น developer, bleach, hair color
2. `เงินทำกำไร` จาก correction และ aftercare เช่น purple shampoo, leave-in
3. `เงินระยะยาว` จากการ lock ร้านให้ใช้ระบบสินค้าเดียวกัน

### คู่แข่งอ้างอิงในตลาดที่เราเห็น

| กลุ่ม | คู่เทียบ |
| --- | --- |
| สีผม / สินค้าร้านซาลอน | Freecia, Gosen |
| developer / bleach แมส | Dcash, Berina |
| salon care | Freecia, Gosen |
| leave-in | GoHair |

### Strategic Gap

ช่องว่างที่น่าสนใจคือ:

> แบรนด์ที่ให้ผลลัพธ์ดูโปรกว่าของแมส แต่ราคาไม่ไกลเกินร้านซาลอนระดับกลาง

นั่นทำให้แบรนด์ของเราควรอยู่ในตำแหน่ง:

`Mid-Pro Salon Brand`

## 3. Value Chain Analysis

### 3.1 Core Chemicals และสินค้าต้นน้ำ

กลุ่มนี้คือหัวใจของธุรกิจร้านทำสี

| หมวด | บทบาท | ความสำคัญ |
| --- | --- | --- |
| สีผม | สร้างเฉดและงานบริการหลัก | สูง |
| ไฮโดรเจนเปอร์ออกไซด์ 3/6/9/12% | ใช้คู่กับสีและฟอก | สูงมาก |
| ไฮโดรเจนเปอร์ออกไซด์ 16% | ใช้ในบางงานเฉพาะ | กลาง |
| ผงฟอก | ยกสีพื้น | สูงมาก |
| เม็ดสีม่วง | แก้เหลือง / tone correction | สูง |
| blue pigment | ลดส้มในงานฟอก | สูงมาก |

### 3.2 Product System ที่ร้านเข้าใจง่าย

ร้านซาลอนไม่ได้ซื้อสินค้าเป็นชิ้นเดี่ยว แต่ซื้อเป็น `ระบบการใช้งาน`

ตัวอย่างระบบ:

- สีผม + developer
- ผงฟอก + developer
- ฟอก + ล้างม่วง
- ทำสี + ดูแลต่อด้วย leave-in

ดังนั้นการขายที่ง่ายที่สุดคือขายเป็น system ไม่ใช่ขายเป็น SKU เดี่ยว

## 4. Bottom-Up Product Opportunity

### 4.1 Hair Color

ข้อดี:

- ตลาดใหญ่
- ใช้ทุกวัน
- ทำให้แบรนด์ดูครบ

ข้อเสีย:

- ต้องมีหลายเฉด
- MOQ สูง
- เสี่ยงเรื่องสีเพี้ยน
- ต้องใช้เวลา build ความเชื่อมั่น

ข้อสรุป:

`ไม่ควรเป็นสินค้าตัวแรก` ถ้าทุนยังจำกัด

### 4.2 Developer (Hydrogen Peroxide 3/6/9/12%)

ข้อดี:

- ใช้คู่กับสีและฟอกเสมอ
- repeat สูง
- ร้านสั่งประจำ
- พัฒนาสูตรให้ต่างได้ด้วยกลิ่นและเนื้อสัมผัส

ข้อเสีย:

- margin ไม่สูงมาก
- ต้องแข่งเรื่องความเสถียรของสูตร

ข้อสรุป:

`เป็นสินค้าบังคับที่ควรมี` เพราะช่วย lock ร้าน

### 4.3 Powder Bleach

ข้อดี:

- เป็น pain point ของร้านชัดมาก
- ร้านยอมเปลี่ยนแบรนด์หากผลลัพธ์ดีกว่า
- มีโอกาสสร้าง hero product

ข้อเสีย:

- ต้องควบคุมสูตรให้ยกดีแต่ไม่ทำผมเสียเกินไป
- ต้องทำ trial หน้างานจริง

ข้อสรุป:

`ควรเป็น category หลักของแบรนด์`

### 4.4 Purple Pigment / Blue Pigment

ข้อดี:

- เป็นจุดต่างที่ลูกค้ารู้สึกได้
- ใช้สร้าง story ทางการตลาดได้ชัด
- ช่วยให้แบรนด์ดู salon-grade มากขึ้น

ข้อสรุป:

- `Purple` เหมาะกับ shampoo / correction
- `Blue pigment` เหมาะกับ hero bleach

### 4.5 Shampoo / Treatment / Conditioner

ข้อดี:

- ต่อยอดจากลูกค้าทำสีได้ง่าย
- ขาย retail ได้
- ช่วยเพิ่มยอดต่อบิล

ข้อเสีย:

- ถ้าไม่มีจุดต่าง จะกลายเป็นสินค้า commodity

ข้อสรุป:

ควรเลือกเฉพาะตัวที่มี story ชัด เช่น `purple shampoo` และ `repair treatment`

### 4.6 Leave-in Conditioner

ข้อดี:

- margin สูง
- เดโมง่าย
- ลูกค้ารู้สึกผลลัพธ์ทันที
- เปรียบเทียบกับ GoHair ได้ตรง

ข้อสรุป:

`ควรเป็น profit SKU ตั้งแต่แรก`

## 6. OEM China Sourcing Model

### โรงงานที่มองหา

โรงงาน OEM ในจีนที่เหมาะควรมีคุณสมบัติ:

- รับสูตร salon line
- ทำ private label ได้
- รับ MOQ ระดับ 1,000-3,000 ชิ้น
- มีตัวอย่างสูตร bleach / developer / purple shampoo / leave-in
- ปรับ fragrance, pigment และ texture ได้

### กลุ่มแหล่งผลิตที่เกี่ยวข้อง

| สินค้า | Supplier ตัวอย่าง | เมือง |
| --- | --- | --- |
| Hair color | Guangzhou Caiqi Cosmetics / Guangzhou Yucaitang / Guangzhou Qiaoyangzi | Guangzhou |
| Developer | Guangzhou Qiaoyangzi / Guangzhou Caiqi | Guangzhou |
| Bleach | Guangzhou Hansiliang / Guangzhou Caiqi | Guangzhou |
| Purple shampoo / Leave-in | Guangzhou Aining | Guangzhou |

### สิ่งที่ต้องถามโรงงาน

- MOQ ต่อสูตรจริงเท่าไร
- MOQ ต่อ packaging เท่าไร
- ราคานี้รวมบรรจุภัณฑ์หรือยัง
- รองรับขนาด 150ml หรือไม่
- มีค่า mold / artwork / label เพิ่มหรือไม่
- เงื่อนไข sample fee
- lead time ผลิตกี่วัน
- export เองได้หรือไม่
- รองรับ formula adjustment กี่รอบ

## 7. Investment Model

### สมมติฐานเบื้องต้น

ผู้ใช้ระบุว่า `eco ในการสั่งต้องเต็มตู้ ราคาประมาณ 300k`

ในทางปฏิบัติ เราควรตีความเป็น 2 ทาง:

1. งบเริ่มต้นรวมการนำเข้าอยู่ราว `300,000 บาท`
2. หรือเป็นงบสำหรับการรวมสินค้าให้คุ้ม logistics ระดับหนึ่ง

ถ้าแบรนด์ยังไม่ใหญ่ การสั่ง `เต็มตู้จริง` อาจหนักเกินไป เพราะจะเพิ่มความเสี่ยงด้าน stock และ cash flow

### โมเดลลงทุนที่เหมาะกว่าในช่วงเริ่ม

`Lean Import Model`

- เปิด 3-4 SKU เท่านั้น
- ใช้ MOQ ต่ำสุดที่โรงงานรับได้
- เน้นสินค้าที่ turnover สูงและเดโมง่าย
- ใช้ร้านทดสอบจริงก่อนขยาย order

### กรอบงบลงทุนเบื้องต้น

| รายการ | งบประมาณโดยประมาณ |
| --- | --- |
| sample + test formula | 20,000-40,000 |
| artwork + packaging setup | 20,000-50,000 |
| มัดจำการผลิตล็อตแรก | 120,000-180,000 |
| shipping / import / ภาษี / เคลียร์ของ | 40,000-80,000 |
| content + sample แจก + sales kit | 20,000-50,000 |
| buffer cash | 30,000-60,000 |

### สรุปงบ

งบ `300,000 บาท` สามารถเริ่มได้ หาก:

- จำกัด SKU
- ไม่เปิดเฉดสีผมจำนวนมาก
- เลือก packaging มาตรฐาน
- ใช้การทดสอบตลาดก่อน reorder

## 9. Product Direction

### 9.1 Blue Bleach

USP:

- ยกสีเร็ว
- ลดส้ม
- ผมไม่แห้งกระด้างเกินไป

สูตรที่ควรมองหา:

- blue pigment
- anti-orange
- dust-controlled
- ถ้ามี bond support จะยิ่งดี

USP:

- creamy
- กลิ่นไม่ฉุน
- ใช้งานคู่กับ bleach ได้เสถียร

ระดับที่ควรทำ:

- 3%
- 6%
- 9%
- 12%

### 9.3 Purple Shampoo

USP:

- ไม่ติดม่วง
- ล้างแล้วโทนใส
- เห็นผลชัดหลังฟอกหรือทำสีอ่อน

### 9.4 Leave-in Conditioner 150ml

USP:

- ซึมเร็ว
- ไม่มัน
- ผมนุ่มทันที
- เหมาะกับการขายต่อจากงานเคมี
